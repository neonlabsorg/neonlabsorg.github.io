How to Swap SPL Tokens

Goal#

To trade an SPL token for another SPL token using the Neon Swap application.

Condition: The SPL token can be any token whose contract complies with the ERC-20 standard and is deployed in Neon EVM.

Before you begin#

  • Understand the following:
    • What is the Solana network.
    • What is the Metamask.
  • The Metamask wallet must be installed on your device and can be accessed from the browser you are using.
  • Learn the guide "How to Transfer SPL Tokens to Neon EVM".

Steps#

The Neon Swap application allows any user to trade one token for another. You just need to connect your wallet to the application following the step-by-step instructions. Swapping tokens will be carried out automatically without sending funds to the exchange. There are some gas transaction fees to pay with the application.

This tutorial outlines the exchange of ETH->USDT tokens.

From the list appeared on the dropdown widow you have to select your wallet (in our case, this is Metamask). Select the account you want to connect to.

Step 2. Select a pair of tokens to be swapped#

After connecting your wallet you can begin setting the trade. You need to select the token that will be swapping from. It will retrive your balance from the connected wallet. In our case, it will be ETH.

If you need another token to exchange, click Select a token to open up a menu where you can select the token for swapping.

If the menu does not contain the token you need, you can specify its symbol or its contract address at the top of the box. But in this case, the token contract must first be deployed on the Neon EVM network.

You can type the symbol of the token in the search box (in our case, the exchange pair is ETH->NEON).

Enter the quantity you want to sell. Before you can send this trade by clicking Swap you need to understand a few things about it:

  • Minimum received - the minimum amount of SOL you will receive if the trade goes through. The trade will fail if price diverges too much during the transaction also known as slippage.
  • Price impact tells you how much your trade will affect this token's price. This value can be seen as aloss you will take on the trade. The closer this number is to zero the better.
  • Liquidity Provider Fee - the fee you will make this trade. Uniswap charges a 0.3 percent fee on the total amount of the trade. This fee is expressed in the token you are selling (In this case ETH).

Once you are ready to perform the trade click Swap.

A confirmation window will pop up with details of the trade. Click Confirm Swap. This will open up a MetaMask window where you can confirm the transaction. You can see estimate of how much gas you will pay

Step 10. Additional settings for trading#

Click the cogwheel icon. The window Transaction Settings will pop-up on the screen to set up the advanced trade:

Transaction settings:

  • Slippage tolerance ‚Äî the allowable amount of price change. It means that the order is fulfilled at the price different from what you expected. If it exceeds the tolerance level selected, the transaction will not go through. If you are trading during high volatility, you may want to increase this value. Increasing the transaction speed can help you to reduce the chances of high slippage.
  • Transaction deadline ‚Äî allows the transaction to automatically cancel if it is taking too long to process.
  • Toggle Expert Mode ‚Äî disables the confirmation screen and removes slippage limits. It is not encouraged you to enable this mode unless you fully understand the risks.
  • Disable Multihops ‚Äî disables complex conversion paths. For example, if there is not enough liquidity in the "A->B" token pair, then the algorithm can involve other tokens to find the "A->C->B" intermediate route and make an exchange for you. This action requires more network fees, because several smart contracts are involved.
